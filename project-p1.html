포용적 AI 에이전트
벤치마크 설계

Organization   Naver Cloud · HyperCLOVA X Period   2025.12 — Now

AX 확산 과정에서 소외되기 쉬운 산업·연령층을 문제로 설정하고, 기존 tau² Agentic Tool Use 벤치마크에 포용성 개념을 주입·평가하는 프레임을 재구축한 프로젝트.

Agentic AI Benchmark LLM-as-a-Judge Inclusive Design
Key Features01
  • 국내 산업 통계 분석을 통해 고령자 비중이 높은 농업·요양·유통·물류 4대 산업을 선정하고, 현장 중심 Real-World Task를 정의
  • 실데이터 분석을 기반으로 고령 사용자 특성을 반영한 페르소나 주입형 사용자 시뮬레이션 환경 구축
  • API 호출 · 도구 사용 · 문제 해결 · 포용성 전반을 평가하는 스코어링 체계 설계
My Role & Contributions02
  • User Simulator 구현 (고령자 페르소나 주입) — 고령자 발화 패턴을 모듈화하고, 발생 확률을 제어하는 로직으로 시뮬레이터의 현실성·재현성 확보
  • 패턴 적용 전·후 발화를 메타데이터로 기록해 패턴별 효과 분석 설계
  • 포용성 평가(Evaluator) 설계 — tau² reward 체계를 확장해 Uncertainty / Pressure / Alienation / Invalidation 4축 기반 평가 프레임 설계
  • 평가축별 점수·근거를 생성하는 LLM-as-a-Judge 스코어링 로직 구현, JSON 스키마 정형화 및 재평가/후처리 파이프라인 구축
  • 농업(농기계) 도메인 Agent Workflow Policy 개선, 한·영 데이터셋 구축, 유통(편의점) 도메인 Tool API 리팩토링
